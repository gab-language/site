This tour covers the core concepts of Gab, more in depth than we've seen so far. The tour assumes some familiarity with programming, but no prior experience with Gab or any particular programming paradigm.

At the end of the tour, you will understand all the features of the language. With this understanding,  you should have a clear idea of what Gab programs look like. 

The best way to follow along is with `gab repl` open in a terminal beside you.
