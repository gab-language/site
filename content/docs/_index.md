# Overview
Welcome to Gab's documentation! Here you will find a brief overview of each of the sections available to you.

### The *Gabonomicon*
The canonical gab handbook. Everything you need to get off the ground and writing Gab lives in this section.

### Modules
This section contains specifications for the modules that come with Gab - specifically, all the modules in the `gab-language/cgab` package.
Such packages include `Numbers`, `Strings`, and `Io`.
These pages are generated from `Specs`, which is itself a builtin module for defining message contracts.

### Protocols
This section contains specifications for the protocols that come with the builtin `gab-language/cgab` package. Such protocols include `seqable` and `streamable`, which
define interfaces for iterating and IO respectively.
